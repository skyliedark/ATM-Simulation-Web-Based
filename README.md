# ATM Simulation (Web-Based)

A simple web-based ATM simulation built with HTML, CSS, and JavaScript.

## Features
- Simulates basic ATM functions (balance check, withdrawal, deposit).
- Clean, responsive UI.

## Technologies Used
- **HTML** (5.3%)
- **CSS** (39.0%)
- **JavaScript** (55.7%)

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
